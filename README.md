# Digital Store

A Digital Store é uma loja virtual desenvolvida como um projeto prático do curso de Desenvolvimento Web Full Stack, promovido pelo Geração Tech da IEL Ceará. Esse projeto foi criado com o objetivo de demonstrar, na prática, os conhecimentos adquiridos ao longo das aulas, especialmente na área de Front-End. Durante o desenvolvimento da loja, aplicamos conceitos fundamentais como estruturação com HTML, estilização com CSS, interatividade com JavaScript e uso de frameworks modernos. A experiência proporcionou uma vivência real de como funciona o processo de criação de uma interface web, desde o planejamento até a entrega final, consolidando o aprendizado de forma prática e colaborativa.
